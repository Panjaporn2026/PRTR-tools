📬 Invoice Send Status Dashboard

สรุปสถานะการส่งใบแจ้งหนี้จาก getinvoice.net — ส่งแล้ว / เปิดอ่านแล้วหรือยัง / ยังไม่ได้ส่ง / ส่ง Error
📘 วิธีการใช้งาน
1. เข้า getinvoice.net → Send to Buyer ตั้งช่วงวันที่ที่ต้องการ แล้วกด Export เลือกฟอร์แมต Excel
2. อัปโหลดไฟล์ .xlsx ที่ได้มาที่นี่ (รองรับหลายไฟล์ จะรวมข้อมูลให้อัตโนมัติ)
3. ระบบจะจัดกลุ่มแต่ละใบเป็นหมวดจากคอลัมน์ Status/Action ในไฟล์ export โดยอัตโนมัติ
⚠️ เกณฑ์การจัดกลุ่ม: Error = Status Failure หรือมี Action "bounce" (แม้ Status จะเป็น Success) | ยังไม่ได้ส่ง = Status In Progress/Not Send (ระบบยังไม่ส่ง) | เลือกไม่ส่ง = Status Don't Send (พนักงานกดเลือกเองว่าไม่ต้องส่ง แยกจาก "ยังไม่ได้ส่ง") | ส่งแล้ว-เปิดอ่านแล้ว/ยังไม่เปิด = Status Success แยกตามว่า Action มีคำว่า "open" หรือไม่
4. (ไม่บังคับ) อัปโหลดไฟล์ Invoice list.xlsx เพิ่ม เพื่อตรวจครบทุกช่องทางตามคอลัมน์ "Send email by": ใบที่ต้องส่งผ่าน Get-invoice จะจับคู่สถานะจากไฟล์ข้อ 1-3 ให้, ใบที่ต้องส่งผ่าน Outlook จะขึ้นเป็นรายการเตือน (ระบบตรวจสอบอัตโนมัติว่าส่งแล้วหรือยังไม่ได้ เพราะไม่มีข้อมูลนี้ในไฟล์)
1. ไฟล์ Export จาก getinvoice.net (Send to Buyer)
2. ไฟล์ Invoice list.xlsx (ไม่บังคับ — เพื่อตรวจครบทุกช่องทาง รวม Outlook)

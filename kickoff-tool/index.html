📋 Kick-off Meeting Note Filler

กรอกไฟล์ ★KICK OFF MEETING NOTE.xlsx (ชีต Master for inovice) โดยไม่ทำลาย dropdown
📘 วิธีการใช้งาน
1. อัปโหลดไฟล์ ★KICK OFF MEETING NOTE - <Client>.xlsx (เปล่าหรือกรอกไปแล้วบางส่วนก็ได้)
2. เปิดหน้า CRM Backoffice โปรเจคของคุณเองในอีกแท็บ แล้วคัดลอกข้อมูลมากรอกในฟอร์มด้านล่าง
3. ช่องที่มีข้อมูลอยู่แล้วในไฟล์ (ไม่ว่างหรือไม่ใช่ "- Select -") จะไม่ถูกเขียนทับ แม้จะกรอกในฟอร์มก็ตาม
4. กด "สร้างไฟล์" แล้วดาวน์โหลด — ไฟล์อื่นๆ ในเวิร์กบุ๊ก รวมถึง dropdown ทุกตัว จะไม่ถูกแตะต้องเลย
